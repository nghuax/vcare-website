# VCare Foundation Plan (Phase 1)

## Scope
Set up a production-style foundation for VCare using Next.js 15, TypeScript, Tailwind CSS, shadcn-style UI primitives, and Prisma + PostgreSQL.

## Steps
1. Establish scalable project structure for web, patient portal, admin dashboard, server logic, and shared contracts.
2. Build shared layout shell and reusable UI components (header, sidebar, card, badge, empty/loading/table states).
3. Configure route groups for public, patient, and admin areas with placeholder pages and clear TODO markers.
4. Set up Prisma schema with initial healthcare coordination model placeholders and environment template.
5. Document setup/run workflow in README and validate with lint/build checks.

## Out of Scope (This Phase)
- Business logic implementation (prescription workflows, reminder jobs, insurance APIs)
- Authentication/authorization integration
- Real external integrations (pharmacies, insurance providers, SMS, push notifications)
